LINEAR PROGRAMMING PROBLEM

Many business and economic situations are concerned with a problem of planning activity. Linear programming is a technique for determining an optimum schedule of interdependent activities in view of the available resources.

Linear Programming Problem

A.Linear programming problem (LPP) consists of three components, namely the decision variable (activities), the objective (goal), and the constraints (restrictions).

1. Decision variable

The decision variable refer to the activities that are competing one another for sharing the resources available.

2. The Objective

A linear programming problem must have an objective which should be clearly identifiable and measurable in quantitative terms.

3. The Constraints

Such constraints must be expressed as linear inequalities or equalities in terms of decision variable.

Basic Assumptions

The following four basic assumptions are necessary for all linear programming problem.

(a) Certainty

In all LPP's it is assumed that all the parameters.

(b) Divisibility (or continuity)

Such variable are not divisible and hence are to be assigned integer values.

(c) Proportionality

This requires the contribution of each decision variable in both the objective function and the constraints to be directly proportional to the value of the variable.

(d) Additivity

The amount of a resource consumed by A and B must be equal to the sum of resources used for A and B individually.
