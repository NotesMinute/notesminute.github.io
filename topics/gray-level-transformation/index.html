Gray level Transformation

1.It is also called by the name Intensity level Transformation. 2.It is one of the simplest image processing techniques. 3.There are three kinds of gray-level transforms as follows.(Fig 28.1) a.Linear Transformation. b.Logarithmic Transformation. c.Power-law Transformation

Linear Transformation

a.It includes simple identity transformation and negative transformation.

Identity Transformation

a.It is denoted by the straight line as in the illustration.(Fig.28.2) b.In it's transition,each value for each input image is directly mapped each other value of output image. c.Which results in same input and output image and so called by the name identity transformation.

Negative transformation

a.It is the second kind of linear transformation. b.It is the invert of identity transformation. c.In it each value of the input image is subtracted from (L-1) and mapped as output image as shown in illustration (Fig 28.3) s=(L-1)r say,L=256 Therefore, S=255-r (for 8bpp image)

Logarithmic Transformation

a.It includes two types of transformation,Log transformation and Inverse transformation.

Log Transformation

1. S = C log(r+1) Where, S and r are input and output image in pixels and C is a constant (for adjustment in log transform). c.A value of 1 is added to each of the pixel of input image, so if there is a pixel intensity of 0 in the image,then log(0)~=Infinity.Adding,1 makes it a minimum value.

Inverse Log Transformation

a.It is the opposite of Log Transform.

Power-Law Transformation

a.It is further classified as Nth Power and Nth root transformation. S=C r^Y Where, Y-->Gamma Function(different displays have different Y value, for Y->1.8 to 2.5(CRT Monitor's).
