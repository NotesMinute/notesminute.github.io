Architecture and its operations 8085 MP

8085 is an 8-bit , general purpose microprocessor.

Arithmetic and Logic unit

It is used to perform mathematical operations like : addition , multiplication , subtraction , division , etc.,

Flag Register

It is an 8-bit register that stores either 0 or 1 depending upon which value is stored in the accumulator.

Accumulator

Used to perform I/O arithmetic and logical units.

General purpose Registers

There are 6 general purpose registers. Can hold 8 bit values.

Program Counter

Hold the address value of the memory to the next instruction that is to be executed. 16 bit Register.

Stack pointer

Works like Stack. 16 bit Special Register.

Temporary Register

8 bit register. Holds data values during arithmetic and logical operations.

Instruction Register and Decoder

8 bit register. Holds the instruction code that is being decoded. Fetched from the memory of instruction.

Timing and Control Unit

Comes under the CPU section , Controls the flow of data from CPU to other devices.

Interrupt Control

There are 5 interrupt signals in 8085 microprocessor. INTR , TRAP , RST7.5 , RST6.5 , RST5.5

Address bus and data bus

Data bus is bidirectional and carries the data which is to be stored. Address bus is undirectional and carries location where data is to be stored.

Serial Input / Output Control

It controls the serial data communication by using serial input data and serial output data.
