TRANSPORTATION METHOD IN ASSIGNMENT PROBLEM

Since an assignment problem is a special case of the transportation problem, it can be solved by transportation method discussed. However, every basic feasible solution of a general assignment problem having a square payoff matrix of order n should have m+n-1 = n+n-1 = 2n-1 assignment or basic cells. But due to the special structure of this problem, any basic solution cannot have more than n assignment. Thus, the assignment problem is inherently degenerate.
