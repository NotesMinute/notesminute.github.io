Pattern types,materials,allowance in production technology

Pattern is a replication of the object to be cast..

Different types of patterns:

1) Single piece pattern 2) Split piece pattern 3) Loose piece pattern 4) Gated pattern 5) Match pattern 6) Sweep pattern 7) Cope and drag pattern 8) Skeleton pattern 9) Shell pattern 10) Follow board pattern

Single piece pattern:

1.This is the simplest type of pattern, exactly like the desired casting, 2.Used for producing a few large castings, for example, stuffing box of steam engine.

Split piece pattern:

These patterns are split along the parting plane to facilitate the extraction of the pattern out of the mould before the pouring operation.

Loose piece pattern:

Loose piece of a pattern which is attached to the pattern during molding, remains in the mold during lift-off and is only then removed separately.

Gated pattern:

A gated pattern is simply one or more loose patterns having attached gates and runners.

Match plate pattern:

1.It is split pattern, 2.Cop and Drags on the opposite site of the metallic (generally) plate, 3.The gates are runners are on the match plate, 4.Can be used for large number of casting with very little hand work, 5.A match plate can be single pattern or a combination of many small patterns.

Sweep pattern:

This type of pattern is used when a casting of large size is to be produced in a short time.

Cope and drag pattern:

1.A cope and drag pattern is a split pattern having the cope and drag portions each mounted on separate match plates. 2.Used when in the production of large castings

Skeleton pattern:

1.This type of pattern is also used in pit or floor molding process. 2.Just like sweep patterns, these are simple wooden frames that outline the shape of the part to be cast and are also used as guides by the molder in the hand shaping of the mould.

Shell pattern:

shell-mold casting, is an expendable mold casting process that uses resin covered sand to form the mold.

Follow board pattern:

1.Consists of tool that is a simple wooden board, 2.Wooden board is used as a base in follow board pattern for molding process

Different Types of Pattern Allowances:

1.Shrinkage or contraction allowance. 2.Draft or taper allowance. 3.Machining or finish allowance. 4.Distortion or camber allowance. 5.Rapping allowance.

Shrinkage or contraction allowance.

Shrinkage is defined as reduce the dimension of casting during solidification or during cooling. There are three types of shrinkage. 1. Liquid Shrinkage 2. Solidification Shrinkage 3. Solid Shrinkage
