Digital computers

Analog and digital signals, Logic gates~ Logic circuits~ Multiplexers and demultiplexers~ Encoder and Decoders~ Flip-flops~ Registers.

Prathipa Ravichandran

Analog signals
A continuous varying signal.

Digital signals
A signal that has only two discrete values.

Logic gates
AND, OR~ NOT~ NAND~ NOR~ EX-OR~ EX-NOR.

Combinational logic circuits
Logic gates are connected together to produce a specified output for specified input variables with no storage elements.

Sequential circuits
The logic circuits whose outputs at any instant of time depends not only on the preent input but also on the past state output.

Half-subtractor
Subtracts two bits.

Full subtractor
Implemented using two half subtractor.

Multiplexer
Several input lines and one output lines.

Demultiplexers
Single input lines and several output lines.

Encoder
2^n input lines and n output lines.

Decoder
n input lines and 2^n output lines.

Flip-Flops
RS, D~ JK~T.

Shift register
Group of flip-flops that can be used to store a binary number.

Shifting types
Serial shifting and Parallel shifting.

Serial shifting
Shifting 1 bit at a time.

Parallel shifting
Shifting all the bits simultaneously.
